\chapter{Sistema IoT de Gestión Polimórfica de Sensores}

\hypertarget{index_intro_sec}{}\doxysection{\texorpdfstring{Introducción}{Introducción}}\label{index_intro_sec}
Sistema de monitoreo de infraestructura crítica que registra, almacena y procesa lecturas de múltiples tipos de sensores (temperatura, presión) de manera unificada utilizando programación orientada a objetos avanzada.\hypertarget{index_features_sec}{}\doxysection{\texorpdfstring{Características Principales}{Características Principales}}\label{index_features_sec}

\begin{DoxyItemize}
\item {\bfseries{Polimorfismo}}\+: Gestión unificada de diferentes tipos de sensores mediante clase base abstracta
\item {\bfseries{Listas Enlazadas Genéricas}}\+: Almacenamiento dinámico de lecturas con plantillas (templates)
\item {\bfseries{Comunicación Serial}}\+: Recepción de datos desde ESP32 por puerto serial
\item {\bfseries{Gestión Manual de Memoria}}\+: Implementación de Regla de los Tres/\+Cinco
\item {\bfseries{Menú Interactivo}}\+: Interfaz de usuario para operaciones CRUD de sensores
