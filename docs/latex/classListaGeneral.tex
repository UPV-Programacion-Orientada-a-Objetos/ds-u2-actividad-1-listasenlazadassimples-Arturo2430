\doxysubsection{Detailed Description}
Lista enlazada de gestión polimórfica para todos los sensores del sistema. 

Mantiene una lista de punteros a \doxylink{classSensorBase}{Sensor\+Base}, permitiendo almacenar diferentes tipos de sensores y procesarlos de manera unificada mediante polimorfismo. Se encarga de liberar la memoria de los sensores al destruirse.
